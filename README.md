# Diploma Notes

A GitHub Pages-ready static website for Kerala Polytechnic / Diploma students.

## What is included

- Separate Revision 2021 and 2015 Materials areas
- Department pages for Revision 2021
- Dynamic subject cards from `assets/js/subjects.js`
- Open Syllabus, View Lessons, and Sample Question Paper actions on every subject card
- 2015 material links extracted from the supplied 2015 reference file
- Mobile responsive HTML, CSS, and JavaScript only

## Model question paper URL generation

Sample question paper links are generated from each subject code:

`https://www.sitttrkerala.ac.in/index.php?r=site%2Fdiploma-modelqp-courses-show&course=SUBJECT_CODE`

Update `assets/js/subjects.js` to add more subjects, revisions, departments, or codes.
