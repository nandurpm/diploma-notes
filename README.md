# Diploma Notes

A GitHub Pages-ready static website for Kerala Polytechnic / Diploma students.

## Features

- HTML, CSS, and JavaScript only
- Dynamic subject cards from assets/js/subjects.js
- Course-code based syllabus links
- Course-code based model question paper links
- Responsive student-friendly UI
- Ready for future custom domain through CNAME

## Model Question Paper URL Pattern

https://www.sitttrkerala.ac.in/index.php?r=site%2Fdiploma-modelqp-courses-show&course=SUBJECT_CODE

Update subject data in assets/js/subjects.js to add more subjects.
