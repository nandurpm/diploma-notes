Course Objectives

  • Use graphics as an engineering language to express ideas and instructions.
  • Use drafting instruments, sketching skills and BIS drawing conventions.
  • Draw and read engineering curves, projections and dimensioning styles.
  • Develop projections of points and lines.
  • Understand orthographic projections and sectional views of objects.
  • Visualize actual objects from engineering drawings.
  • Experiment with simple CADD commands for two-dimensional plane figures.

How to Study

  • Keep one separate drawing notebook for construction steps.
  • Practise with instruments: mini drafter, set squares, compass and scale.
  • Write the given data first, then draw reference line XY.
  • Use thin construction lines and darken only final object lines.
  • Redraw each problem once without looking at the solved sheet.

Revision

One-page Quick Revision

Last-minute Engineering Graphics checklist for the REV2021 1005 drawing examination.

Important Definitions

  • Projection: representation of an object on a plane.
  • Orthographic view: front, top or side view drawn on projection planes.
  • Sectional view: view obtained after cutting an object by a section plane.
  • Isometric view: pictorial view where three axes are equally inclined.
  • Dimensioning: showing size and location details on a drawing.

Procedures to Remember

  • Start projections with XY reference line.
  • Use first angle projection unless the question says otherwise.
  • Use hidden lines only where invisible edges must be shown.
  • In sectional views, hatch only the cut material area.
  • For isometric, keep vertical axis vertical and other two axes at 30°.

Common Exam Mistakes

  • Using dark construction lines.
  • Forgetting projection symbols and view labels.
  • Mixing first angle and third angle placement.
  • Missing center lines for circles/cylinders.
  • Dimensioning inside the object when avoidable.

Last-minute Checklist

Sharp pencils, eraser, compass, scale and set squares ready.
Object lines thick, construction lines thin, hidden lines dashed.
All views aligned with projectors and labelled properly.
Dimensions, arrows, extension lines and units are clear.
