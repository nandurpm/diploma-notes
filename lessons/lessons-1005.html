  • Familiarize the graphics language used in engineering work.
  • Develop drafting and sketching skills and awareness of Indian Standards.
  • Translate ideas into sketches and draw/read curves, projections and dimensioning styles.
  • Develop projection of points and lines, orthographic/sectional views and simple CADD commands.

Key Points Bank

Key Points Bank for Engineering Graphics

Use this as the last-minute memory page before exams and practical sessions.

Visible outline

Thick continuous line

Where used: Object boundary

Common mistake: Do not use construction line thickness.

Projection line

Thin continuous line

Where used: Transfer points between views

Common mistake: Keep light and clean.

Hidden edge

Dashed thin line

Where used: Invisible edge

Common mistake: Do not overuse when sectional view is used.

Dimension line

Thin continuous with arrowheads

Where used: Size indication

Common mistake: Do not cross dimension lines unnecessarily.

First angle projection

Object lies between observer and plane

Where used: Kerala diploma drawings often specify this

Common mistake: Place views in correct relative positions.

Isometric axes

One vertical and two 30° axes

Where used: Isometric drawing

Common mistake: Do not draw horizontal edges as horizontal unless required.
