Truth table format

Inputs -> Output

Meaning: Proves gate/circuit function.

Where used: All digital experiments.

Common mistake: Missing input combinations.

TTL supply

VCC = +5 V, GND = 0 V

Meaning: Required for 74xx ICs.

Where used: Logic gate experiments.

Common mistake: Wrong supply voltage or floating inputs.

Universal gates

NAND/NOR can realize any logic

Meaning: Implement logic using only NAND/NOR.

Where used: Universal gate experiment.

Common mistake: Not tying unused inputs properly.

Counter modulus

Number of output states

Meaning: Counter design and verification.

Where used: Counter experiments.

Common mistake: Wrong reset logic.

Seven segment

BCD/count output -> display segments

Meaning: Visual output for counter.

Where used: Decade/up-down counter.

Common mistake: Wrong common anode/cathode assumption.
