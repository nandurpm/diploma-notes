Mini Formula Bank

Mini Formula Bank - Electronic Circuits

Quick reference cards for revision. Use this only after reading the module explanation.

Voltage gain

Av=Vo/Vi

Meaning: Amplifier output/input voltage ratio.

Where used: CE amplifier problems.

Common mistake: Mixing voltage gain and current gain.

Gain in dB

Av(dB)=20log10Av

Meaning: Logarithmic gain representation.

Where used: Multistage amplifier calculations.

Common mistake: Using 10 log for voltage gain.

Bandwidth

BW=fH-fL

Meaning: Frequency range with acceptable gain.

Where used: Frequency response.

Common mistake: Not locating half-power frequencies.

Resonance

fr=1/(2π√LC)

Meaning: Tuned circuit resonant frequency.

Where used: Tuned amplifiers and LC oscillators.

Common mistake: Wrong units for L and C.

Barkhausen criterion

Aβ=1 and phase shift = 0°/360°

Meaning: Condition for sustained oscillations.

Where used: Oscillator explanation.

Common mistake: Writing only gain condition without phase.

RC phase shift oscillator

f=1/(2πRC√6)

Meaning: Oscillation frequency for equal R and C sections.

Where used: RC oscillator design.

Common mistake: Forgetting √6.

Wien bridge oscillator

f=1/(2πRC)

Meaning: Frequency for equal R and C.

Where used: Audio oscillators.

Common mistake: Using RC phase shift formula.

Duty cycle

D=(TON/T)×100%

Meaning: Percentage of ON time in pulse waveform.

Where used: Multivibrator questions.

Common mistake: Not converting to percentage.
