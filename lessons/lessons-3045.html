Key Points Bank

C Programming Quick Reference

Basic Structure

#include <stdio.h> int main(){ return 0; }

Operators

Arithmetic, relational, logical, increment/decrement, conditional, assignment and bitwise operators.

Arrays

Same type elements stored in continuous memory. Index starts from 0.

Pointers

A pointer stores the address of another variable. Use & for address and * for value at address.

Strings

Character array ending with null character. Common functions: strcpy, strcat, strlen, strcmp.

Functions

Reusable blocks with declaration, definition, call, arguments and return type.

Module 1

Introduction to C Programming Concepts

Topics: C program structure, keywords, variables, constants, data types, type qualifiers, input/output functions, operators, assignment, conditional operator and type casting.

Learning Outcome

Use basic C programming elements correctly and write simple input/output programs.

Simple Explanation

A C program is made of header files, main function, declarations, statements and return value. Variables store data, constants remain fixed, and operators process values.

Important Terms

TermMeaning
KeywordReserved word with predefined meaning.
VariableNamed memory location whose value can change.
Data typeSpecifies type and size of data.
Type castingConverting one data type into another.
Common mistakes: missing semicolon, using = instead of ==, forgetting & in scanf, wrong format specifier.

Module 2

Control Structures, Loops and Arrays

Topics: if, if-else, nested if, switch, while, do-while, for, nested loops, break, continue, goto, one-dimensional arrays, two-dimensional arrays and matrix operations.

Core Idea

Control structures decide the path of execution. Loops repeat statements. Arrays store multiple values under one name.

A pointer variable stores memory address. It helps efficient data access and supports call by reference.

Module 4

Functions in C

Topics: definition, types, advantages, structure of user-defined functions, function declaration, call, arguments, return type, call by value, call by reference and recursion.

Why Functions?

Functions divide a large program into smaller reusable blocks. They improve readability, testing and debugging.

TypeMeaning
No argument, no returnFunction performs task only.
Argument, no returnInput is passed but no value returns.
Argument with returnMost common useful form.
Recursive functionFunction calls itself.

Master Answer Key

Answers for Expected Questions

M1-Q1: A C program contains preprocessor directives, main function, declarations, executable statements and return statement.
M1-Q2: Keyword is reserved word. Variable is named memory location. Constant is fixed value.
M1-Q3: printf displays output and scanf reads input using format specifiers.
M1-Q4: Operators perform arithmetic, comparison, logical, assignment, bitwise and conditional operations.
M1-Q5: Read two values, calculate +,-,*,/ and print results.
M1-Q6: Type casting converts data type, for example converting integer value to float for decimal division.
M2-Q1: if-else tests conditions; switch selects among constant case labels.
M2-Q2: while tests before loop, do-while tests after loop, for loop combines initialization, condition and update.
M2-Q3: Compare a,b,c using if-else and print the largest value.
M2-Q4: Start with 0 and 1, then repeatedly add previous two terms.
M2-Q5: Array operations include insertion, deletion, searching, sorting and traversal.
M2-Q6: Read two matrices, add corresponding elements and display result matrix.
M3-Q1: Pointer stores address. Syntax: int *p;
M3-Q2: Pointer arithmetic includes increment, decrement, addition, subtraction and comparison based on data type size.
M3-Q3: Store base address of string in char pointer and print characters until null character.
M3-Q4: gets reads a string line and puts displays a string line.
M3-Q5: strlen length, strcpy copy, strcat concatenate and strcmp compare strings.
M4-Q1: Function is reusable program block. Advantages: modularity, reuse, debugging and readability.
M4-Q2: Function has return type, name, parameter list, body and return statement if required.
M4-Q3: Call by value passes copy. Call by reference passes address and can modify original value.
M4-Q4: Create function max(a,b,c) and return greatest using if conditions.
M4-Q5: Recursive function calls itself; factorial is common example.
