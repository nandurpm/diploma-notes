Lab record format

Aim -> Algorithm/Flowchart -> Program -> Output -> Result

Meaning: Standard C lab record structure.

Where used: Every C lab experiment.

Common mistake: Writing program without algorithm or output.

Basic arithmetic program

+ - * /

Meaning: Perform operations on two numbers.

Where used: Module 1 experiment.

Common mistake: Integer division mistake.

Decision statement

if / if-else / nested-if / switch

Meaning: Select execution path.

Where used: Largest/smallest and vowel/consonant programs.

Common mistake: Using = instead of ==.

Loop statement

for / while / do-while

Meaning: Repeat code.

Where used: Fibonacci/series programs.

Common mistake: Wrong termination condition.

Array operation

read -> process -> display

Meaning: Use list of values.

Where used: Sum, average, sort and search.

Common mistake: Array index out of range.

String functions

strcpy, strcat, strlen, strcmp

Meaning: Manipulate strings.

Where used: String experiments.

Common mistake: Not including string.h.

Function

prototype -> call -> definition

Meaning: Reusable logic.

Where used: Matrix diagonal/column sums.

Common mistake: Mismatched return type.
