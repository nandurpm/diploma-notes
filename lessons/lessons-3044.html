Mini Formula Bank

Mini Formula Bank - Digital Electronics

Quick reference cards for revision. Use this only after reading the module explanation.

1s complement

Invert every bit

Meaning: Used to represent negative binary numbers.

Where used: Binary subtraction basics.

Common mistake: Forgetting to add 1 for 2s complement.

2s complement

1s complement + 1

Meaning: Standard signed binary representation.

Where used: Binary arithmetic.

Common mistake: Ignoring fixed bit-width.

SOP form

OR of product terms

Meaning: Boolean expression from 1 outputs.

Where used: K-map simplification.

Common mistake: Mixing SOP and POS.

POS form

AND of sum terms

Meaning: Boolean expression from 0 outputs.

Where used: K-map/product simplification.

Common mistake: Grouping 1s instead of 0s for POS.

K-map grouping

Groups of 1,2,4,8... cells

Meaning: Simplifies Boolean expression.

Where used: Up to 4 variable K-map.

Common mistake: Making diagonal groups or non-power-of-two groups.

Full adder

S=A⊕B⊕Cin, Cout=AB+BCin+ACin

Meaning: Adds three binary bits.

Where used: Adder design.

Common mistake: Wrong carry expression.

Counter modulus

Modulus = number of states

Meaning: Counting sequence length.

Where used: Counter design.

Common mistake: Not resetting at correct count.

Shift register

Stores and shifts bits

Meaning: Serial/parallel data movement.

Where used: Sequential circuits.

Common mistake: Confusing SISO/SIPO/PISO/PIPO.
